# JARVIS Voice Assistant

JARVIS es un asistente de voz personalizado que puede realizar varias acciones según los comandos de voz que reciba. A
continuación se detallan las funcionalidades que puede realizar, junto con las frases específicas que el usuario puede
decir para activar estas acciones.

## Funcionalidades

### 1. Activar el Protocolo de Buenos Días

**Acción**: Encender un enchufe Meross.

- **Frases de Activación**:
    - "Inicia el protocolo buenos días"
    - "Activa el protocolo buenos días"

### 2. Activar el Protocolo de Buenas Noches

**Acción**: Apagar un enchufe Meross.

- **Frases de Activación**:
    - "Inicia el protocolo buenas noches"
    - "Activa el protocolo buenas noches"

### 3. Obtener la Hora Actual

**Acción**: Informar la hora actual.

- **Frases de Activación**:
    - "¿Qué hora es?"
    - "Dime la hora"

### 4. Información del Próximo Gran Premio de Fórmula 1

**Acción**: Informar sobre el próximo Gran Premio de Fórmula 1.

- **Frases de Activación**:
    - "¿Cuándo es el próximo gran premio de Fórmula 1?"
    - "¿Cuándo es el siguiente gran premio de Fórmula 1?"

### 5. Información del Nombre del Propietario

**Acción**: Decir el nombre del propietario.

- **Frases de Activación**:
    - "¿Cuál es mi nombre?"
    - "Dime mi nombre"

### 6. Actualizar Datos del Propietario

**Acción**: Actualizar la información del propietario de manera interactiva.

- **Frases de Activación**:
    - "Actualiza mis datos"
    - "Modifica mis datos"

### 7. Despedirse y Detener el Bot

**Acción**: Despedirse y detener la ejecución del bot.

- **Frases de Activación**:
    - "Apágate"
    - "Apagar"
    - "Detente"

### 8. Información del Propietario

**Acción**: Informar sobre los detalles del propietario.

- **Frases de Activación**:
    - "¿Quién es el propietario?"
    - "¿Cómo me llamo?"

### 9. Contar una Muerte Matavise

**Acción**: Contar una historia de muerte al estilo Matavise.

- **Frases de Activación**:
    - "Dime una muerte"
    - "Cuéntame una muerte de Matavise"

### 10. Reiniciar el Equipo

**Acción**: Reiniciar el equipo.

- **Frases de Activación**:
    - "Reinicia el ordenador"
    - "Reiniciar el equipo"
    - "Reiniciar el sistema"

### 11. Cambiar la Palabra Clave

**Acción**: Cambiar la palabra clave de activación.

- **Frases de Activación**:
    - "Cambiar palabra clave"

### 13. Inciar un recordatorio

**Acción**: Iniciar un recordatorio para recordar al usuario que haga alguna acción.

- **Frases de Activación**:
    - "Recuérdame que ponga una lavadora en 5 minutos"
    - "Recuérdame que haga los deberes en 2 horas"

## Ejemplo de Uso

Para iniciar el protocolo de buenos días, el usuario podría decir:

    "Inicia el protocolo buenos días"

Para obtener la hora actual, el usuario podría decir:

    "¿Qué hora es?"

Para actualizar los datos del propietario, el usuario podría decir:

    "Actualiza mis datos"

Para reiniciar el ordenador, el usuario podría decir:

    "Reinicia el ordenador"

## Notas Adicionales

- El bot JARVIS utiliza reconocimiento de voz para detectar los comandos del usuario.
- Asegúrese de que el micrófono esté bien configurado y en una ubicación adecuada para minimizar el ruido de fondo.
- La precisión del reconocimiento de voz puede variar según el entorno y la claridad del habla del usuario.
- Es recomendable utilizar frases completas y claras para asegurar que el bot entienda correctamente los comandos.
